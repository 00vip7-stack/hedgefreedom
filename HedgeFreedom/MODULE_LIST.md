기존 확장/세부 모듈 설명은 아래 유지

## 세부 기능/확장 모듈
- API 연동 모듈 (예: 서버 데이터 자동 불러오기, REST API 통신)
- 엑셀 업로드/파싱 모듈 (excel-upload.html, xlsx.js 활용)
- 엑셀 집계/자동 집계 모듈 (내부/외부 집계, 실시간 테이블 반영)
- 외부 데이터 연동 모듈 (금융기관, 환율, 금리 등)
- 사용자 인증/권한 모듈 (로그인, 관리자모드 등)
- 알림/에러처리 모듈 (실시간 피드백, 오류 안내)
- UI/UX 공통 모듈 (메뉴바, 테이블, KPI 위젯 등)
- 문서/가이드 모듈 (사용법, 정책, 컨설팅 안내 등)
- 기타: 신규 기능/세부 기능은 별도 파일로 분리하여 모듈화 권장

## 설명
- 각 모듈은 ERP 스타일로 독립적으로 동작하며, 메뉴에서 상호 연결됩니다.
- 사소한 기능도 별도 모듈로 분리해 확장성 있게 관리합니다.
- 신규/세부 모듈 추가 시 이 목록에 계속 업데이트하세요.
